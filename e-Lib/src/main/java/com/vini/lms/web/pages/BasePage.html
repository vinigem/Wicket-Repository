About

e-Library allows the users to search for books and issue a book online
from anywhere and anytime.

We have a large collection of books under various categories. This web site provides the users
an easy way to search for their books and issue accordingly.

This Web site is user friendly and has all the functionality of a Library.

The users can login to their account if they are registered and view their profile and books
they have issued in past.

New Users can register and become a member and avail the facility of this Online Library.
